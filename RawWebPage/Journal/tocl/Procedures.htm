ACM Transactions on Computational Logic (TOCL)

Procedures for Submitting Accepted Papers

Typesetting of accepted Papers

Please prepare your manuscript using LaTeX, and follow ACM's Guidelines for Submitting Accepted Articles. If your paper has an electronic appendix then format it as explained in Section 8 of the instructions.

Electronic submission of accepted papers

There are two sets of procedures for submitting accepted papers for final publications.

  • Papers submitted to TOCL after 1 September 2010 are using the Manuscript Central tracking system and that system will provide authors with instructions for final submission.
  • Papers submitted to TOCL prior to 1 September 2010 are using the TOCL Editorial Office. In this case, follow the instructions below.

Submissions of accepted papers via the TOCL Editorial Office

For submissions that were not submitted via Manuscript Central: The preferred form of submitting the files is by sending to the TOCL Editorial Office, acm-tocl@cs.utexas.edu, the URL address from which they can be fetched.

In the same email please send the precise address and email of the corresponding author. He/she will be in charge of the galley proofs.

Please send the URL's of the following files:

  • the PDF file that corresponds to the manuscript,
  • the single LaTeX2e file (so the .bbl file needs to be included in the .tex file), and the EPS files for all graphics (if there are many files in this group, please send a URL of a single gzipped tar file instead).

What happens after you submitted the files to the TOCL Editorial Office?

  • The PDF file will be placed in the list of Accepted Papers for TOCL.
  • The copyedited the manuscript will be sent from the ACM Headquarters to the corresponding author for possible corrections.
  • The corresponding author will be asked to process these corrections in his/her LaTeX files and resubmit them to the TOCL Editorial Office in the same way as originally.
