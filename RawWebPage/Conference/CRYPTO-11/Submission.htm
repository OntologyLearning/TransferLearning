Instructions for Submitting Camera-Ready Versions
of Accepted Papers
Use the camera-ready revision form to submit the camera-ready version. You will need the submission-ID and password that were given to you when you first submitted the paper. (Please contact the Program Chair if you have lost your submission-ID or password.) The deadline for uploading the files is May 27, 2011, 23:59 UTC; this is a firm deadline.

The proceedings of CRYPTO 2011 will be published in Springer′s Lecture Notes in Computer Science (LNCS). Authors must prepare their camera-ready version in LaTeX2e. You should obtain the LNCS LaTeX2e class file (llncs.cls), which can be obtained off Springer's site at:

http://www.springer.de/comp/lncs/authors.html
Click on the link "Information for LNCS Authors", which will take you to a page with a zip file that includes the class file, and some documentation on how to use that class file.

The page limit for your camera-ready paper is 18 pages total, including bibliography and all appendices. Please prepare an archive file (either zip, tar, tar.gz, or tgz) that includes the following files:

  • The source LaTeX file, including any figures, style files, and all other files that are needed to produce the camera ready copy.
  • A PDF file that was created from the source LaTeX file(s).
Then upload the archive file to
https://secure.iacr.org/websubrev/crypto2011/submit/cameraready.php

Authors must also download the IACR copyright form from

http://www.iacr.org/forms/
then print it out and sign it. The signed form must either be sent by FAX to
IACR Copyright Forms Registry
+1 (815) 572-9739
or be scanned into PDF format and sent as an attachment by email to
To: copyrightform@iacr.org
Subject: CRYPTO 2011 copyright form for "<title of my paper>"
The form should be FAXed or emailed by May 27, 2011.
